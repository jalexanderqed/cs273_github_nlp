## Reproduction Environment

The issue was reproduced with the version : 4.1.0, 4.5.1
MacBook Pro(Retina, 15-inch, Mid 2014)
OS X El Capitan
Version 10.11.1
## Reproduction Steps
### Step 1 : Install sbt and scala.

```
brew install sbt
brew install scala
```
### Step 2 : Clone scalechain.

```
git clone https://github.com/ScaleChain/scalechain
```
### Step 3 : Switch branch to proto-stream

```
cd scalechain
git checkout proto-stream
```
### Step 4 : run unit tests.

```
sbt clean test
```
